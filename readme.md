# Neovim Configuration by tanvir

Personal Neovim setup for:

- Python
- R
- Lua
- Bash

Features:

- LSP
- Autocomplete
- Treesitter
- Telescope
- Neo-tree
- ToggleTerm
- Git integration
- Formatting
- Debugging
- Session management

## Installation

```bash
git clone https://github.com/USERNAME/nvim-config.git
cd nvim-config
./install.sh
```
